
安装tk
```
>> sudo apt install python3-tk  (Ubuntu)
>> yum install python3-tk    (Centos)
>> sudo chown -R $(whoami) /usr/local/share/man/man8    (MacOS)
>> brew install tcl-tk    (MacOS)
>> echo 'export PATH="/usr/local/opt/tcl-tk/bin:$PATH"' >> ~/.zshrc    (MacOS)
>> source ~/.zshrc    (MacOS)
```


安装tk开发类库:
```
>> sudo apt install tk-dev  (Ubuntu)
>> yum install tk-devel    (Centos)
>> sudo chown -R $(whoami) /usr/local/share/man/man8    (MacOS)
>> brew install tcl-tk-devel    (MacOS)
>> echo 'export PATH="/usr/local/opt/tcl-tk/bin:$PATH"' >> ~/.zshrc    (MacOS)
>> source ~/.zshrc    (MacOS)
```
